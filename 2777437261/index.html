Mysql查询空格相关问题

发表于 2022-08-18 分类于 Mysql 阅读次数： Valine：

上述 4 句 sql，其中前三句 sql 的效果是一致的，mysql 等值匹配的时候会忽略后置的空格，数据库中的记录如果 name 有’jt’，’jt ’都会被查询出来。而前置空格则会精确匹配。

同时 mysql 默认是大小写不敏感的。’jt’和’JT’或者’Jt’都是一样能被‘jt’等值匹配出来。

如果想要精确匹配，则可以通过 where name like ? 或者 where name = binary ?来实现
